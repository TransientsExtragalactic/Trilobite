triceratops.models.generic.light_curve.FRED.VARIABLES#

FRED.VARIABLES: tuple[ModelVariable, ...] = (ModelVariable(name='t', base_units=''),)#

The model’s variables.

Each element of VARIABLES is a ModelVariable instance that defines a single variable of the model. These variables represent the inputs to the model that can vary during evaluation. Each variable contains information about the base units, name, etc. of the variable. Notably, variables differ from the parameters (PARAMETERS) in that variables are do NOT have default values, do NOT have validity ranges, and are expected to be provided at model evaluation time.

Type:

tuple of ModelVariable
