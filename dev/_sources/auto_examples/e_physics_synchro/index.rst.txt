********************************************
Physics Backend Examples - Synchrotron
********************************************

In this example gallery, we'll showcase the :mod:`radiation.synchrotron` module and its capabilities.
For detailed documentation regarding synchrotron radiation physics, please refer to the
:ref:`synchrotron_theory`. For notes on the synchrotron API and how to use all of the various
functions, please refer to :ref:`synchrotron_core`, :ref:`synchrotron_microphysics`, :ref:`synchrotron_seds`, and :ref:`synchrotron_cooling`.
