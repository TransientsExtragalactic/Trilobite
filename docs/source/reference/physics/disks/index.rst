=========================================
Accretion Disks
=========================================

- blurb about the relevance of accretion disks for Triceratops. The need to model disk emission for inverse compton,
  state transitions for outflows, long-lived UV emission, etc.
- Many implementations of disks with different behaviors, physics, etc. Some are more phenomenological, some are more physical.
- The goal in Triceratops to provide a flexible framework for comparatively light-weight disk models (no GRMHD, no ray tracing, etc.)
  that can be used for a variety of applications. The framework is designed to be extensible, so that more complex models can be added in the future.
